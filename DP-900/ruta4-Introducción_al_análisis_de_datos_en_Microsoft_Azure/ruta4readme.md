# Introducción al análisis de datos en Microsoft Azure

El crecimiento exponencial de los datos en los últimos años está impulsando la transformación digital de empresas y otras organizaciones, al facilitar la toma de decisiones rápidas e informadas mediante el análisis de datos.

Microsoft Azure ofrece múltiples servicios que se pueden combinar para crear soluciones analíticas a gran escala que aprovechan las últimas tecnologías y técnicas para la ingesta, el almacenamiento, el modelado y la visualización de datos.

Esta ruta de aprendizaje te ayuda a prepararte para la certificación **Azure Data Fundamentals**.
